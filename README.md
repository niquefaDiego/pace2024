# PACE 2024

Entry for the [PACE challenge 2024](https://pacechallenge.org/2024).

## Requirements

- `python3`
- `pip`
- `venv`
- `gcc`

## Set up

Install `pace2023verifier` CLI tool:

- `python3 -m venv python_venv`

## Run

- `source python_venv/bin/activate`
- `python3 -m pip install -r requirements.txt`
- `./run.sh`
- `./grade.sh`
