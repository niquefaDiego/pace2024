# PACE 2024

Entry for the [PACE challenge 2024](https://pacechallenge.org/2024).

## Requirements

- `python3`
- `pip`
- `venv`
- `gcc`

## Set up

Install `pace2023verifier` CLI tool (using venv fron convenience):

- `python3 -m venv python_venv`
- `source python_venv/bin/activate`
- `python3 -m pip install -r requirements.txt`

## Run and grade solution

- `source python_venv/bin/activate` (Only needed once per session)
- `./run.sh`
- `./grade.sh`
